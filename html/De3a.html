What is Ajax?

As defined in wikipedia

AJAX is a group of interrelated web development techniques used to create interactive web applications or rich Internet applications. With Ajax, web applications can retrieve data from the sever asynchronoursly in the background without interfering with the display and behaviour of the existing page.

Brief History

Year Event
1995 Java applets were introduced the first way to load data from client side asynchronously
1996 IFrame element were introduced, which also have this functionality
1999 Microsoft created the XMLHTTP ActiveX control in Intemet Explorer 5 which then followed by other browsers as XMLHttpRequest object
2005 The term "Ajax" was borned
2006 The W3C in order to to make it official web standard, publicized the first draft specification for XMLHttpRequest Object
